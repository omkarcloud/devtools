## 😎 Run it Locally 

1️⃣ Clone the Magic 🧙‍♀️:
```shell
git clone https://github.com/omkarcloud/devtools
cd devtools
```
2️⃣ Install Dependencies 📦:
```shell
npm install --legacy-peer-deps
```

3️⃣ Run it 😎:
```shell
npm run dev
```
